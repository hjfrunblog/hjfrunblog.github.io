# 基础概念

## 成交量

成交量三律

- 卖在买先
- 价在量先
- 庄在散先

## 7 条铁律

### k 线形态口诀

- 连续小阳出大阳
- 连续小阴出大阴
- 连续大阳出卖点
- 连续大阴出买点

### 强势股特征

红肥绿瘦

### 关注涨停股

涨停股就是潜力股，

- 没有涨停过的票不买
- 没有放量的票不买
- 没有突破的票不买

### 短线不恐高

- 低位股横盘浪费时间
- 想快速获利就不要怕高位，选择龙头股控制仓位，顺应趋势

### 短线追求新鲜

- 有新题材就做新题材
- 有新龙头就看新龙头

新龙头，新题材新热点才是短线的机会；

### 只做上升股

只做趋势，不做抄底，顺势而为才是王道

- 上升趋势的股票回调时是进场的机会
- 下降趋势的反弹大多是陷阱

### 反听消息

盘口消息，专家观点，新闻媒体，股吧消息，反着听胜算更高；
