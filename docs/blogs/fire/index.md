# FIRE

什么是 **FIRE**?

即 `Financial Independence, Retire Early`.

敬畏市场，市场永远没有错！！！错的是自己，反省自身！！！

财务自由，提前退休。

这是我在 40 岁之前的目标。

我之前的工作是在马士基当一名 `Senior Software Engineer`。这份工作，老实说时间自由，主要是居家办公。待遇也算过得去。收入一年有超过 60w，也算是很多人羡慕不已的了。但是这些年我一直有些迷茫。我觉得没有成就感，没有激情。做一天和尚，撞一天钟。对新的技术不再有激情，不再愿意花很多时间对新技术学习研究。可能年龄上来了，学东西也变得慢了。这些年一直在一线当大头兵，写代码，没有做什么管理。在外企这边，做了很多东西，从前端到后端，到运维。见识的东西很多，但是都好像没有很多深度，每天浑浑噩噩的完成`jira`上的几个`issue`，就把一天的活干完了，没有什么太有难度的任务。这个项目上学到的东西，到了下一个项目，似乎变得一文不值。自己也一天天的蹉跎时间，像是在被温水煮青蛙。

今年 6 月中旬，一天中午。睡午觉起来。葡萄牙领导跟我约了一个会议。还加上了 HR。顿时有种不妙的感觉。居然是裁员！！！
