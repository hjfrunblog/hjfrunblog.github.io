# 开窍

## 几句话

- 炒股什么都不要看，看势：拥抱趋势，趋势为王。永远不要用价格的高低决定自己的买入卖出的条件；
- 降低自己的操作频率：学会等待：等主线题材的爆发；只要主流题材爆发，重仓参与；强者恒强，做板块内最强；
- 短线的本质是试错：寻找大趋势
  - 胜率其实不重要，阴亏比才是关键；亏损是交易的必然成本；
  - 高手与低手的关键比拼是在逆势的时候谁亏得少，而不在顺势的时候，谁挣得多；
- 想在市场长久地活着，时而小赚，时而小亏，偶尔大赚，绝不大亏。严格按照交易计划止盈止损；
- 不计划不交易：在盘后去计划你的交易，在盘中去交易你的计划
  - 一流的方法，三流的执行 --> 亏损
  - 三流的方法，一流的执行 --> 盈利
- 买入分歧，卖出一致
  - 什么叫分歧？预期差。
  - 当一个强势的股票出现分歧的时候，就是买点，分歧的表现就是连续放量上张，突破箱体，然后股价开始出现滞涨或者调整；说明股票的多孔双方分歧比较大；出现分歧之后，没有跌破重要的支撑位；比如十日线箱体下沿等等。这个时候，可以考虑低吸买入，后续还会继续上涨；
  - 当创出新高，一致加速，所有散户都觉得还能继续上涨的时候，就是卖点。这个叫卖出一致。
- 不做下跌趋势的股票，只做在上升通道中的个股
  - 炒股就是个概率游戏，尽量选择赢面大的
- 短线最难的是**选股**

  - 最近 10 个交易日，没有涨停的不看
  - 最近一个月没有放量突破过箱体的不看
  - 题材没有想象空间的不买
  - 没有板块效应的也不买

- 题材是第一生产力
- 板块效应决定持续性
- 股性决定主力
- 趋势决定未来

## 看形态捕捉起涨点

1. 涨停倍量长上影，仙人指路往上行

   > 当股票涨停后，可能会出现回调；或者次日直接出现冲高回落的仙人指路形态；仙人指路是股票在低位出现长上影线；实体很小，影线是实体的两倍以上；这个时候下方成交量出现倍量，这个走势其实是主力的试盘动作；主力通过拉升股价，来试探上方的抛压情况；一旦出现这种情况，通常预示后面的股价会上涨；可以考虑在回调时择机介入

2. 回调不破涨停低，一边拉升一边洗

   > 实体涨停后开始回调；但是回调过程中，收盘价没有跌破涨停板的最低价；涨停板代表的是主力的行为；其最低价是一个重要的支撑位；只要没有被跌破，说明市场多放力量较强；在看量价关系，回调时缩量，上涨时放量，这是典型的主力洗盘手法；一般情况下，洗盘时间比较短，等后市再次出现放量阳线，说明主力洗盘结束。

3. 涨停过后连探底，潜底试盘要注意

   > 当股票涨停后开始回调，但是回调过程中有长长的下影线，跌幅不大，小幅探底；主力通过这种方式测试下方的支撑力度和筹码是否松动；如果试盘阶段出现缩量，说明抛压很轻，支撑力度较强；如果试盘阶段，收盘价跌破涨停板的最低价或重要均线，可能有主力出货的嫌疑；等待后续成交量放大，股价快速回升，说明主力试盘结束。

4. 连阳倍量再回踩，放量反包就起涨

   > 当股票在低位出现连续阳线，成交量持续放大，随后股票出现缩量下跌。回踩下方的关键支撑位。确认支撑有效。回踩时出现缩量；反包时出现放量；两者相辅相成，缺一不可；当抛压释放完毕，止跌企稳后，出现放量反包前面的洗盘阴线，说明主力洗盘结束；
